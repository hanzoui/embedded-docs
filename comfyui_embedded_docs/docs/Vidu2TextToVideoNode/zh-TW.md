Vidu2 文字轉影片生成節點能根據文字描述建立影片。它連接到外部 API，根據您的提示生成影片內容，讓您可以控制影片的長度、視覺風格和格式。

| 參數 | 資料類型 | 必填 | 範圍 | 描述 |

| `prompt` | STRING | 是 | - | 用於影片生成的文字描述，最大長度為 2000 個字元。 |
| `duration` | INT | 否 | 1 到 10 | 生成影片的長度，單位為秒。可使用滑桿調整數值（預設值：5）。 |
