此節點使用變分自編碼器（VAE）將壓縮的音訊表示（潛在樣本）轉換回音訊波形。它透過處理較小且重疊的區塊（tiles）來管理記憶體使用，使其適合處理較長的音訊序列。

| 參數名稱 | 資料類型 | 必填 | 數值範圍 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `samples` | LATENT | 是 | 不適用 | 待解碼音訊的壓縮潛在表示。 |
| `vae` | VAE | 是 | 不適用 | 用於執行解碼的變分自編碼器模型。 |
| `tile_size` | INT | 否 | 32 至 8192 | 每個處理區塊的大小。音訊將以此長度分段解碼以節省記憶體（預設值：512）。 |
| `overlap` | INT | 否 | 0 至 1024 | 相鄰區塊重疊的樣本數。這有助於減少區塊邊界處的失真（預設值：64）。 |
