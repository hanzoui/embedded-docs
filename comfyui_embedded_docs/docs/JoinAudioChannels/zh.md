此节点将两个独立的单声道音频输入合并为一个立体声音频输出。它接收左声道和右声道，确保它们具有兼容的采样率和长度，并将其合并为一个双声道音频波形。

## 输入参数

| 参数 | 数据类型 | 必填 | 取值范围 | 描述 |
|-----------|-----------|----------|-------|-------------|
| `audio_left` | AUDIO | 是 | | 用作结果立体声音频中左声道的单声道音频数据。 |
| `audio_right` | AUDIO | 是 | | 用作结果立体声音频中右声道的单声道音频数据。 |

**注意：** 两个输入音频流必须是单声道（单通道）。如果它们具有不同的采样率，较低采样率的声道将自动重采样以匹配较高的采样率。如果音频流长度不同，它们将被修剪至较短音频流的长度。

## 输出

| 输出名称 | 数据类型 | 描述 |
|-------------|-----------|-------------|
| `audio` | AUDIO | 生成的立体声音频，包含合并后的左声道和右声道。 |
