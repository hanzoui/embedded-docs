Узел ColorToRGBInt преобразует цвет, заданный в шестнадцатеричном формате, в одно целочисленное значение. Он принимает строку цвета, например `#FF5733`, и вычисляет соответствующее целое число RGB, комбинируя компоненты красного, зеленого и синего.

## Входные параметры

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `color` | STRING | Да | N/A | Значение цвета в шестнадцатеричном формате `#RRGGBB`. |

**Примечание:** Входная строка `color` должна быть ровно 7 символов в длину и начинаться с символа `#`, за которым следуют шесть шестнадцатеричных цифр (например, `#FF0000` для красного). Узел выдаст ошибку, если формат некорректен.

## Выходные параметры

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `rgb_int` | INT | Рассчитанное целочисленное значение RGB. Оно выводится по формуле: `(Красный * 65536) + (Зеленый * 256) + Синий`. |
