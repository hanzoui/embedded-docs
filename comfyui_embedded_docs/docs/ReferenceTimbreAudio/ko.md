이 노드는 "ace step 1.5" 프로세스에서 사용할 참조 오디오 음색을 설정합니다. 컨디셔닝 입력과 선택적으로 오디오의 잠재 표현을 받아, 해당 잠재 데이터를 컨디셔닝에 첨부하여 워크플로우의 후속 노드에서 사용할 수 있도록 합니다.

## 입력

| 매개변수 | 데이터 타입 | 필수 | 범위 | 설명 |
|-----------|-----------|----------|-------|-------------|
| `conditioning` | CONDITIONING | 예 | | 참조 오디오 정보가 첨부될 컨디셔닝 데이터입니다. |
| `latent` | LATENT | 아니요 | | 참조 오디오의 선택적 잠재 표현입니다. 제공되면 해당 샘플이 컨디셔닝에 추가됩니다. |

## 출력

| 출력 이름 | 데이터 타입 | 설명 |
|-------------|-----------|-------------|
| `conditioning` | CONDITIONING | 수정된 컨디셔닝 데이터입니다. 선택적 `latent` 입력이 제공된 경우, 이제 참조 오디오 음색 잠재 표현을 포함하고 있습니다. |
