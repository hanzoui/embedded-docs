Узел Empty Ace Step 1.5 Latent Audio создает пустой латентный тензор, предназначенный для обработки аудио. Он генерирует "тихий" аудиолатент заданной длительности и размера пакета, который может использоваться в качестве отправной точки для рабочих процессов генерации аудио в Hanzo Studio. Узел вычисляет длину латента на основе введенных секунд и фиксированной частоты дискретизации.

## Входы

| Параметр | Тип данных | Обязательный | Диапазон | Описание |
|-----------|-----------|----------|-------|-------------|
| `seconds` | FLOAT | Нет | 1.0 - 1000.0 | Длительность генерируемого аудио в секундах (по умолчанию: 120.0). |
| `batch_size` | INT | Нет | 1 - 4096 | Количество латентных изображений в пакете (по умолчанию: 1). |

## Выходы

| Имя выхода | Тип данных | Описание |
|-------------|-----------|-------------|
| `LATENT` | LATENT | Пустой латентный тензор, представляющий тихое аудио, с идентификатором типа "audio". |
