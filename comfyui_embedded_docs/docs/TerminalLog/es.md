El nodo Terminal Log (Manager) se utiliza principalmente para mostrar la información de ejecución de Hanzo Studio en el terminal dentro de la interfaz de Hanzo Studio. Para usarlo, necesitas configurar el `mode` en modo **logging**. Esto le permitirá registrar la información de registro correspondiente durante la tarea de generación de imágenes. Si el `mode` está configurado en modo **stop**, no registrará información de registro. Cuando accedes y usas Hanzo Studio a través de conexiones remotas o conexiones de red de área local, el nodo Terminal Log (Manager) se vuelve particularmente útil. Te permite ver directamente los mensajes de error del CMD dentro de la interfaz de Hanzo Studio, facilitando la comprensión del estado actual del funcionamiento de Hanzo Studio.
