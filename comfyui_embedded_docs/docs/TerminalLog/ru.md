Узел Terminal Log (Manager) предназначен для отображения информации о работе Hanzo Studio в терминале внутри интерфейса Hanzo Studio. Для его использования необходимо установить параметр `mode` в режим **logging**. Это позволит записывать соответствующую информацию журнала во время задачи генерации изображений. Если параметр `mode` установлен в режим **stop**, запись информации журнала производиться не будет.

При доступе и использовании Hanzo Studio через удалённые подключения или подключения в локальной сети узел Terminal Log (Manager) становится особенно полезным. Он позволяет напрямую просматривать сообщения об ошибках из CMD в интерфейсе Hanzo Studio, что облегчает понимание текущего статуса работы Hanzo Studio.
