Terminal Log (Manager) düğümü, öncelikle Hanzo Studio arayüzü içindeki terminalde Hanzo Studio'nin çalışma bilgilerini görüntülemek için kullanılır. Kullanmak için `mode` parametresini **logging** moduna ayarlamanız gerekir. Bu, görüntü oluşturma görevi sırasında ilgili log bilgilerini kaydetmesini sağlayacaktır. Eğer `mode` parametresi **stop** moduna ayarlanırsa, log bilgilerini kaydetmeyecektir.
Hanzo Studio'ya uzak bağlantılar veya yerel ağ bağlantıları üzerinden eriştiğinizde ve kullandığınızda, Terminal Log (Manager) düğümü özellikle kullanışlı hale gelir. Bu düğüm, CMD'den gelen hata mesajlarını doğrudan Hanzo Studio arayüzü içinden görüntülemenize olanak tanıyarak, Hanzo Studio'nin mevcut çalışma durumunu anlamayı kolaylaştırır.
